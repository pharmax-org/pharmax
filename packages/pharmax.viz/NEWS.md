# pharmax.viz 0.2.0-alpha

- Added `px_residuals()` for residual diagnostic panels.
- Added `px_vpc()` for lightweight visual predictive checks.
- Added `px_npde()` for NPDE-style diagnostic panels.
- Added `px_forest()` for parameter and covariate effect forest plots.
- Added `px_plot_grid()` for composing Pharmax plots with shared annotation.
- Added a public/synthetic diagnostic gallery vignette.

# pharmax.viz 0.1.0-alpha

- Added `theme_pharmax()` for publication-ready ggplot2 styling.
- Added `px_colors()`, `scale_color_pharmax()`, and `scale_fill_pharmax()`.
- Added `px_spaghetti()` for individual concentration-time profiles.
- Added `px_gof()` for four-panel goodness-of-fit diagnostics.
- Added `px_save()` for plot export with publication defaults.
- Added testthat tests and generated roxygen documentation.
