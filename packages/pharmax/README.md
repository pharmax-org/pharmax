# pharmax

`pharmax` coordinates the current public proof slice for the Pharmax R package
work.

## Public Packages

```r
library(pharmax)
px_public_packages()
```

The public repository currently includes only:

- `pharmax`
- `pharmax.viz`
- `pharmax.ml`

These packages are exploratory and require human pharmacometric review. They are
not validated for regulatory submission or clinical decision support.
