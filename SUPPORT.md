# Support

`pharmax` is a public proof-slice project. Support is best-effort through
GitHub Issues.

## Where To Ask

- Bug report: open a bug issue.
- Feature idea: open a feature request.
- Documentation problem: open a documentation issue.
- Security concern: follow `SECURITY.md`.

## Before Opening An Issue

Please include:

- package name: `pharmax`, `pharmax.viz`, or `pharmax.ml`
- package version
- R version
- operating system
- minimal reproducible example
- confirmation that the example uses public or synthetic data only

## Not In Scope For Public Issues

Do not post:

- proprietary clinical datasets
- sponsor, employer, or confidential project details
- patient-level or subject-level data
- regulatory submission drafts
- credentials, tokens, or local machine secrets
