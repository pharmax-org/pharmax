pharmax

A conservative public proof-slice for tidy, reproducible R workflows in pharmacometrics. This public repository shows package proof that can be installed, tested, and reviewed without private strategy context.

GitHub pharmax.viz pharmax.ml

Public Package Surface

pharmax

Coordinator package with helpers that describe the current public package surface.

pharmax.viz

Visualization helpers for public and synthetic pharmacometric examples.

pharmax.ml

R-native exploratory ML helpers for synthetic benchmark workflows and human-reviewed evidence summaries.

Install

options(repos = c(
  pharmax = "https://pharmax-org.r-universe.dev",
  CRAN = "https://cloud.r-project.org"
))

install.packages(c("pharmax", "pharmax.viz", "pharmax.ml"))

Direct GitHub installs remain available for source checkout workflows.

pak::pak("pharmax-org/pharmax/packages/pharmax")
pak::pak("pharmax-org/pharmax/packages/pharmax.viz")
pak::pak("pharmax-org/pharmax/packages/pharmax.ml")

Boundary

  • Exploratory package feedback only.
  • Public or synthetic examples only.
  • No autonomous clinical, dosing, or regulatory decision-making.
  • No private planning, business strategy, or product-platform commitments in this repo.
